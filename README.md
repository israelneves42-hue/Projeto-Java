# Projeto Java

Este repositório contém um exemplo simples em Java: `main.java` imprime "Hello, World!".

Como compilar e executar (Windows):

```bash
javac main.java
java main
```

Sugestões úteis:
- Renomeie a classe para `Main` e o ficheiro para `Main.java` (convencional).
- Mova o código para `src/` e use Maven ou Gradle para gerenciar o build.
- Crie um repositório remoto (GitHub/GitLab) e publique o código.

Arquivos adicionados automaticamente: `.gitignore`.
